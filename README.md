## Next.js Personal Resume

My Personal resume website built with Next.js and Typescript and hosted on Vercel. 
The project uses the next/api to host content and serve it based on preffered browser language.

Mathias Oxholm

## Dependencies

- clsx
- gsap
- sass
- next-seo
- typescript
